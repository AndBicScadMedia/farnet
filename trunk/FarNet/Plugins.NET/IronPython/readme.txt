This is a small plugin to run IronPython scripts in FarManager

place your scripts in the 'Scripts' folder and restart FAR

compilation:
  *download IronPython
  *place IronMath.dll, IronPython.dll to bin folder 
  *run build.bat



PS. IronPython is a python implementation for Microsoft .NET
http://www.codeplex.com/IronPython
