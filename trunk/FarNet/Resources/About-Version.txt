Описание
Плагин для написания плагинов для Far Manager на .NET языках.
Использует объектную модель, аналогичную модели плагина SHPCE.
Домашняя страница: http://code.google.com/p/farnet

Description
Plugin for writing pligins for Far Manager in .NET languages.
Implemented object model is similar to model of SHPCE plugin.
Home Page: http://code.google.com/p/farnet
