	= PREREQUISITES =

 * Far Manager 2.0.1641
 * Plugin FarNet 4.3.26
 * .NET Framework 2.0

	= DESCRIPTION =

This editor tool alters default editor actions on Ctrl-Left/Right and
Ctrl-Shift-Left/Right: go to or select up to the next or previous word bound.
